# 커뮤니케이션즈오브코스 (Communications of Course) 홈페이지

## 📌 프로젝트 개요
**커뮤니케이션즈오브코스(CoC)**는 부동산·건설 광고 전문 크리에이티브 에이전시입니다.  
본 홈페이지는 회사 소개, 서비스, 포트폴리오, 클라이언트, 문의 기능을 갖춘 원페이지 웹사이트입니다.

---

## ✅ 구현된 기능

| 섹션 | 내용 |
|------|------|
| **Hero** | 풀스크린 히어로, 파티클 애니메이션, 실적 카운터 |
| **About** | 회사 소개 및 특징 3가지 |
| **Services** | 6가지 서비스 카드 (카피라이팅, 크리에이티브기획, 건설부동산광고, ATL, 디지털마케팅, 브랜드커뮤니케이션) |
| **Portfolio** | 20개 프로젝트, 카테고리 필터(전체/아파트/리조트·호텔/브랜드/상업시설), 더보기/접기 |
| **Clients** | 주요 클라이언트 무한 티커, 업무 프로세스 4단계 |
| **CTA** | 행동 유도 배너 |
| **Contact** | 문의 폼 (DB 저장), 회사 연락처 |
| **Footer** | 브랜드, 링크, 서비스, 연락처, 소셜 |

### 인터랙션
- 헤더 스크롤 시 배경 변경
- 모바일 햄버거 메뉴
- 스크롤 리빌 애니메이션 (IntersectionObserver)
- Hero 파티클(canvas) 애니메이션
- 숫자 카운터 애니메이션
- 포트폴리오 카테고리 필터
- 스크롤 상단 이동 버튼
- 네비게이션 활성 링크 추적

## 🗄️ 데이터 모델

### `inquiries` (문의 테이블)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID |
| name | text | 이름/회사명 |
| phone | text | 연락처 |
| email | text | 이메일 |
| service | text | 서비스 유형 |
| message | rich_text | 문의 내용 |
| submitted_at | datetime | 접수 일시 |

**API Endpoint:** `POST tables/inquiries`
